# Breast Cancer Prediction Using Machine Learning

This project uses machine learning techniques to classify breast tumors as benign or malignant using medical diagnostic data.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Machine Learning Models

- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest
- K-Nearest Neighbors
- Naive Bayes
- Decision Tree


## Results

Support Vector Machine (SVM) and Logistic Regression achieved the highest prediction accuracy and demonstrated strong performance in breast cancer classification.

## Objective

To develop an accurate machine learning model for breast cancer prediction and support early diagnosis through data-driven analysis.
